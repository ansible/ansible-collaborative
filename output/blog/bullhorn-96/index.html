    Gwmngilfen contributed

    Webhooks for Ansible Matrix Rooms

    This is (mostly) news for our various Working Groups! If you're involved in a Working Group and would like to receive notifications in your Matrix room, then rejoice! For we have added the Hookshot bridge to our homeserver, which means we can now relay incoming payloads into a given room. It also supports RSS feeds should you want it.

    If this sounds like it would be of interest to you, then please reach out to me and I'll help you out.

    We have released an Ansible provider for Terraform that can run Ansible playbooks and roles on infrastructure provisioned by Terraform. For more info, see this blog post and the project repo.
